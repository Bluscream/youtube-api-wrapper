This is the API

aboutGet

Retrieves information about the API

URL

http://localhost:7077/about

HTTP Method

GET

Response Type

AboutResponse

Parameters

captionGet

Retrieves captions for a YouTube video

URL

http://localhost:7077/caption

HTTP Method

GET

Response Type

Parameters

  • Query: videoId String

    YouTube video ID

  • Query: format String

    Caption format (e.g., vtt, srt)

  • Query: lang String

    Language code

rootGet

Retrieves comprehensive information about a YouTube video

URL

http://localhost:7077/

HTTP Method

GET

Response Type

YoutubeAPIProxyResponse

Parameters

  • Query: videoId String

    YouTube or pietsmiet.de video ID

  • Query: fresh Boolean

    Invalidates the cache so a fresh result is returned
